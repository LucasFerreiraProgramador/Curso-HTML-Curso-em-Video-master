## Projeto Demonstrativo – Portfólio Freelancer

Autor: **Lucas Ferreira**

Projeto desenvolvido para fins de portfólio, com foco em **HTML, CSS e JavaScript**.
O objetivo é demonstrar estrutura de páginas, organização de arquivos,
uso de mídia (áudio e vídeo) e interações básicas com JavaScript.

### Funcionalidades
- Páginas institucionais
- Estrutura semântica HTML
- Inclusão de mídia (áudio e vídeo)
- Scripts JavaScript básicos

### Tecnologias
- HTML5
- CSS3
- JavaScript

### Observação
Projeto demonstrativo para apresentação a clientes.
